#MEAN全栈javascript开发框架示例

##关于MEAN

* MongoDB是一个使用JSON风格存储的数据库，非常适合javascript。(JSON是JS数据格式)
* ExpressJS是一个Web应用框架，提供有帮助的组件和模块帮助建立一个网站应用。
* AngularJS是一个前端MVC框架。
* Node.js是一个并发 异步 事件驱动的Javascript服务器后端开发平台。

##运行

    1. npm install
    2. npm install -g bower
    3. npm install -g nodemon
    4. npm start
